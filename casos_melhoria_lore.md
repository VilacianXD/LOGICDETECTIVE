# 🕵️‍♂️ Casos com Lore Melhorada (Tema Noir) - Logic Detective

Este arquivo contém o remapeamento completo da lore de todos os 12 casos do jogo para uma atmosfera Noir clássica e coesa. Aqui estão definidos os suspeitos, as vítimas e os dados físicos separados (Sexo, Idade, Altura, Olhos, Cicatrizes) juntamente com uma descrição visual curta em parágrafo para cada personagem, que será exibida ao jogador no jogo. As armas/itens e os locais também contam com características físicas para servirem de base para futuras pistas.

As pistas adaptadas foram removidas a fim de permitir a substituição direta dos elementos do arquivo abstrato.
